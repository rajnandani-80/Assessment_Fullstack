# 🚀 Node.js Backend Assessment Project

This is a backend API project built using **Node.js, Express.js, MongoDB, JWT Authentication, and Multer for file uploads**.  
It provides user authentication, user management, and profile image upload functionality.

---

## 📌 Features

- User Registration (Signup)
- User Login with JWT Authentication
- Protected Route (Get Logged-in User Profile)
- CRUD Operations on Users
- Profile Image Upload using Multer
- MongoDB Database Integration

---

## 🛠️ Tech Stack

- Node.js
- Express.js
- MongoDB + Mongoose
- JWT (Authentication)
- Bcrypt (Password Hashing)
- Multer (File Upload)
